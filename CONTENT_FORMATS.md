# Content Formats

## Purpose

Reusable short-form content formats for the AI YouTube Audience-in-the-Loop channel.

These formats help agents generate consistent videos while keeping the audience in control of the next direction.

## Format Principles

Every format should include:

1. A fast hook.
2. A reference to the audience signal.
3. One clear visual or conceptual payoff.
4. A prompt for the next audience decision.

Default structure:

```text
Hook
  -> audience chose this
  -> show the result
  -> twist or reveal
  -> top comment decides next
```

## 1. Top Comment Builds It

The audience chooses what the AI creates next.

Example:

```text
"Top comment said build a floating city, so the AI started with the impossible part: keeping it in the sky."
```

Best for:

- Objects.
- Worlds.
- Machines.
- Cities.
- Creatures.
- Tools.
- Vehicles.

CTA:

```text
"What should the AI add to it next?"
```

## 2. Top Comment Breaks It

The audience chooses what goes wrong.

Example:

```text
"The community built a floating city. Then top comment asked what would happen if the engines failed."
```

Best for:

- Simulations.
- Engineering drama.
- Story tension.
- Cause-and-effect videos.

CTA:

```text
"Top comment decides how the AI fixes this."
```

## 3. Top Comment Evolves It

The audience chooses the next evolutionary step.

Example:

```text
"Yesterday it was a tiny AI village. Today, top comment evolved it into a living city."
```

Best for:

- Creatures.
- Civilizations.
- Worlds.
- Robots.
- Ecosystems.

CTA:

```text
"What should it evolve into next?"
```

## 4. Top Comment Names It

The audience names a character, place, tool, species, or machine.

Example:

```text
"The community built this machine. Now top comment gets to name it."
```

Best for:

- Audience ownership.
- Recurring characters.
- Worldbuilding.
- Mascots.

CTA:

```text
"Top comment names this thing."
```

## 5. Top Comment Chooses the Next Path

The audience selects between branches.

Example:

```text
"The AI reached a fork: ocean, desert, or orbit. Top comment chose orbit."
```

Best for:

- Adventure chains.
- Fictional simulations.
- Decision trees.
- Choose-your-own-path episodes.

CTA:

```text
"Where should it go next?"
```

## 6. Top Comment Adds a Rule

The audience adds a constraint.

Example:

```text
"Top comment added one rule: the city can never touch the ground."
```

Best for:

- Creative challenges.
- AI experiments.
- Simulation constraints.
- Comedy.

CTA:

```text
"Add one rule for the next episode."
```

## 7. Top Comment Creates the Problem

The audience creates the challenge the AI must solve.

Example:

```text
"Top comment gave the AI one problem: keep a city alive with no sunlight."
```

Best for:

- Puzzle-style videos.
- Engineering concepts.
- Survival scenarios.
- Fictional design challenges.

CTA:

```text
"What problem should the AI solve next?"
```

## 8. Top Comment Creates the Villain

The audience creates an opposing force.

Example:

```text
"The floating city was peaceful until top comment invented the storm serpent."
```

Best for:

- Fictional stories.
- Worldbuilding.
- Creature design.
- Mythic or cinematic arcs.

Guardrail:

Keep villains fictional and non-targeted.

CTA:

```text
"What should the villain do next?"
```

## 9. Top Comment Changes the Simulation

The audience changes one variable in a simulation.

Example:

```text
"Top comment changed gravity to 30%. The AI city immediately got weird."
```

Best for:

- Science-style shorts.
- Simulations.
- Visual experiments.
- Educational hooks.

CTA:

```text
"What variable should we change next?"
```

## 10. Top Comment Unlocks the Next Level

The audience unlocks a new stage.

Example:

```text
"The community survived Level 1. Top comment unlocked the sky biome."
```

Best for:

- Game-like content.
- Progression systems.
- Series arcs.
- Repeatable formats.

CTA:

```text
"What should Level 3 be?"
```

## 11. Top Comment Makes It Cute

The audience steers tone toward cozy, cute, or charming.

Example:

```text
"Top comment said the robot needed a tiny garden. Honestly, correct."
```

Best for:

- Warm content.
- Creature or robot companions.
- Visual delight.
- Brand softening.

CTA:

```text
"What wholesome upgrade should it get next?"
```

## 12. Top Comment Makes It Cosmic

The audience pushes the concept toward scale and wonder.

Example:

```text
"Top comment sent the city into orbit, so now the AI has to build a moon dock."
```

Best for:

- Sci-fi.
- Space visuals.
- Big reveals.
- Cinematic moments.

CTA:

```text
"What cosmic problem should happen next?"
```

## 13. Comment Battle

Two strong audience ideas compete, and the next video tests both.

Example:

```text
"Half of you wanted a dragon. Half wanted a solar engine. The AI combined them."
```

Best for:

- Multiple strong comments.
- Community debate.
- Hybrid ideas.
- Fun tension.

CTA:

```text
"Which side won? Top comment decides what survives."
```

## 14. Community Upgrade

The audience improves the previous build.

Example:

```text
"The first version was unstable, so the comments redesigned it."
```

Best for:

- Iterative builds.
- Engineering.
- Design feedback.
- Before/after reveals.

CTA:

```text
"What upgrade should the community add next?"
```

## 15. The AI Misunderstood

The AI interprets a comment too literally in a funny, safe way.

Example:

```text
"Top comment said give it wings. The AI gave the entire city wings."
```

Best for:

- Humor.
- Weird visuals.
- Light chaos.
- Memorable shorts.

CTA:

```text
"What should the AI misunderstand next?"
```

## Recommended First MVP Formats

Start with these:

```text
Top Comment Builds It
Top Comment Chooses the Next Path
Top Comment Adds a Rule
Top Comment Creates the Problem
Top Comment Evolves It
```

These are easy to explain and repeat.

## Agent Guidance

When generating a video idea, pick one format first.

Do not mix too many formats in one short.

The audience should immediately understand how to participate in the next episode.
